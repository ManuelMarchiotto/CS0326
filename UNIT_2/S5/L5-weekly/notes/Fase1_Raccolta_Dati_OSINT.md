# Fase 1: Raccolta Dati OSINT su Elon Musk

## Introduzione

Questa fase del progetto si concentra sulla raccolta di informazioni pubbliche (OSINT) relative a Elon Musk. L'obiettivo è creare un profilo dettagliato che possa essere utilizzato per progettare uno scenario di attacco di Spear-Phishing personalizzato.

Le informazioni sono state raccolte da fonti pubbliche: articoli di notizie, documenti ufficiali, profili social, database pubblici e annunci aziendali.
